Llamas With Tanks

A game where the player is a llama in a tank firing at other tanks going through levels with various difficulties. Defeat all the enemies and get through all the levels to beat the game! This game is inspired by Wii Tanks and the soul of Hell Divers 2! Spreading democracy with every projectile!

  • Duration Aug 2024 - Nov 2024

  • Team Size 8 People

  • Engine Unreal

  • Role UI/UX Design

My Contributions

My main contribution to the game were making menus and UI elements for everything in the game including transitions. I also created the graphic design for the UI using Photoshop.

Scoreboard Blueprint

I constructed a blueprint that increments the scoreboard by adding 100 points for every enemy destroyed on the field, ensuring that the score dynamically updates during gameplay.

  • Time Spent1 Hour
  • PeopleMe
  • ToolsBlueprint

Description

I constructed a blueprint that increments the scoreboard by adding 100 points for every enemy destroyed on the field, ensuring that the score dynamically updates during gameplay.

Intentions

The intent was to give the player a sense of progress as they played. Having the score increase by 1 wasn't fun so we changed it to 100 instead. The bigger numbers were better for user experience.

Score Board Functionality

The score function adds a integer of 100 every time a enemy is destroyed.

Getting User Experience

I take time to give the game out to others to see what worked and what did not work with the game. Recording there experience and asking questions to get a feel for how the user is feeling about certain mechanics and systems within the game. Along side recording bugs and noting them down for the design team.

What I Learned

This was my first time using Unreal Engine. I was able to learn a lot about Blue Print and how to implement it into UI elements in the game to make sure that the design goal of the UI was met. Here I was able to learn from others who were more experience in engine and I was able to pick that up and ask questions when needed to get the best results for our game.
